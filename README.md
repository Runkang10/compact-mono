# Compact Mono
A library with collection of Kotlin extensions of [Paper API](https://docs.papermc.io/paper/dev/api/) and
implementations for my paper plugins.

## Required Dependencies
- Kotlin **2.4.10**
- Paper API **>=26.1.2**

## Usage
### Gradle
```kotlin
dependencies {
    implementation("io.github.runkang10:compact-mono:2.1.0")
}
```

### Maven
```js
<dependency>
    <groupId>io.github.runkang10</groupId>
    <artifactId>compact-mono</artifactId>
    <version>2.1.0</version>
</dependency>
```
